# Library-Management-System

A simple console-based Library Management System built using Python. This project allows users to manage books in a library through an easy menu-driven interface.

## Features

* Add New Books
* View All Books
* Search Books by ID
* Issue Books
* Return Books
* Delete Books
* View Available Books

## Technologies Used

* Python
* Lists
* Dictionaries

## Purpose

The main goal of this project is to practice Python fundamentals and basic Data Structures concepts such as lists, dictionaries, searching, and CRUD operations.

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run the Python file:

   ```bash
   python main.py
   ```
4. Use the menu options displayed in the console.
